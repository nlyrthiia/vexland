# VexLand: Web-Based Roguelike Dungeon Crawler

## Overview

**VexLand** is a web-based roguelike dungeon crawler built on **Avalanche**, designed to offer an immersive and strategic gaming experience with seamless wallet integration. Originally launched as a **Telegram Mini App**, VexLand has successfully transitioned to a fully web-based platform, supporting both **PC and mobile** users. Players can access VexLand directly through any **web browser** using a EVM-compatible wallet.

## Technical Stack

VexLand is built with modern web technologies on the Avalanche blockchain:

- **Frontend:**
  - React 18, TypeScript, Vite, TailwindCSS
  - `@mysten/dapp-kit` — EVM wallet connection
  - `viem` — EVM client library
  - `@tanstack/react-query` — data fetching
- **Game Engine:** Cocos Creator
- **Smart Contracts:**
  - Solidity (edition 2024.beta)
  - Shared objects, capability-based access control
  - Low gas fees and fast finality

## User Onboarding Guide

### Step 1: Setting Up a Avalanche Wallet

1. Install **MetaMask** ([metamask.io](https://metamask.io)) or another EVM-compatible wallet.
2. Create a new account and securely back up your recovery phrase.
3. Fund your wallet with **AVAX** tokens.

### Step 2: Accessing VexLand

1. Open your browser and visit [vexland.xyz](https://www.vexland.xyz/).
2. Click "Connect Wallet" and select your EVM wallet.
3. Approve the connection and start playing.

### Step 3: Gameplay Mechanics

- Explore dungeons, defeat monsters, and earn rewards.
- Check in daily to earn points (3x with Season Pass).
- Compete on the **leaderboard** for top rankings.
